# Flux 2 Pro Streamlit App

Production-ready Streamlit app that generates branded real estate images with Flux 2 Pro using three image references.

## Streamlit Community Cloud Secrets Setup

1. Open your deployed app in Streamlit Community Cloud.
2. Go to **Settings -> Secrets**.
3. Add the following values and save:

```toml
FAL_KEY = "your_fal_api_key_here"
APP_PASSWORD = "your_app_password_here"
```

After saving, restart the app if prompted.
